FG24361   システム最適化

2.0 単位, 3 年次, 春AB 火1,2
遠藤 靖典

授業概要

システムの表現,評価手法,制約条件が与えられたとき,目的関数を最適化するための基本的な数理的技法(LP,NLP,組み合わせ最適化など)について講義する。

備考

FG34361と同一。
対面

授業方法

講義

学位プログラム・コンピテンスとの関係(学習・教育到達目標との関係)

この科目は、「1.1 論理的・数学的な思考力と解析力」に対応している。

授業の到達目標(学修成果)

(1)最適化問題を表す一般的記号、最適解が存在しない場合、大域的最適解、局所的最適解の理解などを講義と演習を通じて理解すること。
(2)線形計画問題、生産計画問題と栄養問題、図的解法を講義と演習を通じて理解し、計算できること。
(3)シンプレックス解法における生産計画問題の解法、2段階法について、講義と演習を通じて、タブローを作り計算できること。
(4)講義と演習を通じて双対問題の理解、双対問題が作れること、弱双対定理を理解すること。
(5)講義と演習によって凸集合、凸集合でないこと、凸関数、関数が凸でないこと、を比較的簡単な集合と関数について示すことができること。
(6)講義と演習によって2次形式が凸であるための条件を知り、2変数、3変数などの場合について計算できること。
(7)講義と演習によって比較的簡単な関数についてヘッセ行列が計算でき、凸である条件が示せること。
(8)講義を通じてラグランジュ乗数法、クーン・タッカーの条件を知り、演習によって、簡単な問題に適用して解を求めることができること。
(9)講義と演習を通じて、最適化計算法について、代表的な計算法の簡単な説明ができること。
(10)講義と演習を通じて、組み合わせ最適化について、代表的な問題について説明ができること。

キーワード

システム最適化、最適解、凸性、シンプレックス法、ラグランジェの未定乗数法、組合せ最適化

授業計画

教科書に沿って進める。原則として全て対面授業とするが、状況に応じてオンラインに切り替える可能性がある。講義の際は板書のノートをとり、教科書と合わせて理解するのがよい。なお、演習の時間を設ける。

第1回 最適化の概念、様々な最適化問題、数学的準備、記号の導入、最適化問題の記述、大域的最適性と局所的最適性
concept of optimization, various optimal problems,  mathematical preparation, introduction of notations, description of optimal problems, goral/local optimization 		担当： 遠藤 靖典
第2回 線形計画問題、例題と図的解法、線形計画問題の標準形
linear programming problem, graphical solution, standard form of linear programming problems
第3回 線形代数の復習、線形計画法の基本定理
basic theorems of linear programming problem
第4回 シンプレックス法
the simplex method
第5回 2段階シンプレックス法
the two-phase simplex method
第6回 双対性とその意義、内点法の概略
Duality and its significance, outline of the interior point method
第7回 非線形計画問題、凸集合と凸関数
nonlinear programming problem, convex set and convex function
第8回 2次関数の凸性、非負定値行列
convexity of quadratic function, nonnegative matrix
第9回 最適性の条件、ラグランジュ乗数法とクーン・タッカーの条件
optimal condition, Lagrange multiplier method and Kuhn-Tucker conditions
第10回 ニュートン法、組み合わせ最適化、ナップザック問題、巡回セールスマン問題
Newton method, combination optimization, knapsack problem, traveling salesman problem

履修条件

成績評価方法

毎回、授業中に指示されたいくつかの演習問題を解き、提出する。また、期末試験を行う。演習課題の総点数を20%、期末試験を80%とし、合計で60%を取ることを単位取得要件とする。状況により、期末試験に代えて総合レポートを課す場合がある。

学修時間の割り当て及び授業外における学修方法

授業は講述（50%）と演習（50%）とする。

その他(受講生にのぞむことや受講上の注意点等)

授業の狙い：システム設計と開発における最適化の概念はシステム工学の中心的な課題である。この授業を受講し、演習問題を解くことで、線形計画法、非線形計画法、組み合わせ最適化について、基本概念とアルゴリズムについて理解することができる。数学モデルを用いる様々な分野に応用できる基礎能力を有する技術者をめざすため、この科目によって論理的・数学的思考力やコンピュータを利用して情報処理を行う能力を育ててほしい。

関連情報：本講義で対象とする最適化理論は、エネルギー最小化・経路最適化・スケジューリング問題等、工学の様々な面だけでなく、経済学・金融学等の人間の営みのあらゆる面で必要となるので、理解を深めてほしい。

受講生に望むこと：この授業で取り上げる線形計画問題は、線形代数の基本的知識を前提としている。また、非線形最適化においては、線形代数IIIにおける対称行列の固有値問題、解析学における平均値の定理、偏微分などを用いる。単に計算ができるだけでなく、論理的かつ抽象的な思考に慣れることを期待している。予習・復習は必ず行うこと。
