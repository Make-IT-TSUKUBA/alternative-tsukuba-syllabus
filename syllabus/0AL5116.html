授業概要

Building on the fundamentals established in "0AL5115 Special Lecture on Policy and Planning Sciences I," this course provides a more advanced study how modern machine learning is used in data analysis.

The course will initiate with a concise summary of the "0AL5115" course content, ensuring a smooth transition towards more complex topics.

This course will continue to thoroughly investigate several modern algorithms, covering both supervised and unsupervised machine learning. In addition to algorithms learned in "0AL5115", the course will discuss basic neural networks, including deep learning.

A significant feature of the course will be a study of automatic machine learning, a dynamic area which greatly simplifies the use of machine learning algorithms, while preserving high predictive performance.

The course will conclude with principal component analysis, a representative tool of unsupervised machine learning  that could simplify the analysis of complex data sets.

In summary, this course provides an intensive, advanced perspective of machine learning, introducing practical and high-demand skills for data analysis.

授業の到達目標(学修成果)

Students will learn fundamental machine learning algorithms for analysing tabular (cross-sectional) data.

キーワード

Statistical learning; Machine learning; Regression models; supervised and unsupervised learning.

授業計画

The course will cover the following topics.

1) A quick recap of topics learned previously in "0AL5115":
• machine learning versus traditional data analysis,
• supervised and unsupervised learning,
• cross-validation, and its use for hyperparameter selection,
• regression trees,
• tree-based ensemble algorithms (bagging, random forests, boosting),
• standard sequence in machine learning analysis.
2) Neural networks and deep learning:
a. single layer neural networks,
b. multilayer neural networks,
c. fitting a neural network.
3) Automatic machine learning:
a. minimum setup to run an automatic machine learning,
b. introduction to H2O, an AutoML data interface,
c. benefits of automating machine learning analysis.
4) Unsupervised learning:
a. difficulties of unsupervised machine learning,
b. basics of principal component analysis (PCA),
c. major applications of PCA.
5) Course summary.

履修条件

This course is designed to build on the knowledge gained from "0AL5115". Although previous enrollment in "0AL5115" is not mandatory, it would greatly enhance your learning experience. If the contents of "0AL5115" is unfamiliar, I strongly encourage to diligently review the main topics of "0AL5115", using the following chapters from the course textbook (“An Introduction to Statistical Learning with Applications in R”, for more details, see below):

    Chapter 2: "Statistical Learning"
    Chapter 3: "Linear Regression"
    Chapter 5: "Resampling Methods"
    Chapter 8: "Tree-based methods"

It's crucial that you familiarize yourself, at the very least, with the materials provided in the computer lab sections of these textbook chapters.

For data analysis, software R will be used. To prepare for using R, download a free version of RStudio Desktop (https://posit.co/download/rstudio-desktop/) using default settings (just click on " Next" without changing default options shown during setup).
